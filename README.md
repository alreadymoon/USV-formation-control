# USV-formation-control
Simulation codes for "Event-Triggered Optimal Line-of-Sight Formation Control for Underactuated USVs with Unknown Disturbance"
The full code will be made available upon acceptance/publication of the paper.
For inquiries, please contact: 1732597142@hrbeu.edu.cn

## File Description

### Main Scripts
| File | Description |
|------|-------------|
| `main_unified_ETM.m` | Main simulation script for the proposed ETO-LOS scheme. Initializes USV states, runs the two-stage ACI control loop, implements the unified event-triggered mechanism, maps control outputs to actuator commands, and generates all simulation figures. Sampling interval: 0.05 s. |
| `main_channel_ETM.m` | Comparative simulation script using the channel-wise fixed-threshold ETM baseline, where independent triggering conditions are assigned to the surge force and yaw moment channels separately. |

### USV Dynamic Models

| File | Description |
|------|-------------|
| `haitun2_1.m` | Dynamic model of Follower 1 (Dolphin 2 USV). Implements 3-DOF nonlinear ship dynamics with hydrodynamic forces, actuator time delay (first-order filter, time constant 0.1 s), and wave disturbance injection from towing tank-measured RAO/QTF data. |
| `haitun2_2.m` | Dynamic model of Follower 2. Same structure as `haitun2_1.m`; implemented as a separate function due to MATLAB persistent variable constraints. |
| `haitun2_3.m` | Dynamic model of the Leader USV. Same structure as `haitun2_1.m`. |

### Control Modules

| File | Description |
|------|-------------|
| `ACI1.m` | First-stage Actor-Critic-Identifier network for the kinematic layer. Estimates the unknown leader-follower interaction term $\Delta_i$ and computes the optimal virtual control law $\hat{\alpha}_i^*$. |
| `ACI2.m` | Second-stage Actor-Critic-Identifier network for the dynamic layer. Estimates the unknown lumped dynamics $f(\nu_i)$ including hydrodynamic forces and wave disturbances, and computes the optimal control input $\hat{\bar{\tau}}_i^*$. |
| `leader_path_reference.m` | LOS-based path following guidance for the leader USV. Generates desired surge velocity $u_d$, yaw rate $r_d$, and reference position along a parameterized path (sinusoidal, straight line, or circular). Uses a virtual target point with longitudinal error compensation. |
| `leader_control_law_PD.m` | PD controller for the leader USV to track the reference path generated by `leader_path_reference.m`. |

### Actuator Mapping

| File | Description |
|------|-------------|
| `force_to_actuator.m` | Maps controller output $[\tau_u, \tau_r]$ to physical actuator commands: propeller speed $n$ (rps) and rudder angle $\delta$ (rad). Solves for $n$ numerically via thrust coefficient curve $K_T = k_0 + k_1 J_p + k_2 J_p^2$, and resolves $\delta$ analytically. |
| `actuator_to_force.m` | Maps actuator commands $[n, \delta]$ back to the actual forces acting on the hull, accounting for propeller open-water characteristics and wake fraction correction. Used inside the USV dynamic model to simulate realistic actuator nonlinearity. |

### Utility Functions

| File | Description |
|------|-------------|
| `Lidar.m` | Computes LOS range $\rho_{ij}$ and angle $\lambda_{ij}$ between follower and leader from relative position. |
| `euler2.m` | First-order Euler integration utility. |
| `ssa.m` | Small angle wrapping function to keep heading angles within $(-\pi, \pi]$. |
| `draw_boat.m` | Visualization utility for drawing USV icons on trajectory plots. |

### Data Files

| File | Description |
|------|-------------|
| `wave_fixed90.mat` | Time-domain wave disturbance signal under 90° beam sea condition, synthesized from towing tank-measured RAO and QTF data of the Dolphin 2 platform via the ITTC wave spectrum. Will be released upon acceptance. |
| `result_unified.mat` | Saved simulation results for the proposed Unified ETM. Will be released upon acceptance. |
| `result_channel.mat` | Saved simulation results for the Channel ETM baseline. Will be released upon acceptance. |

### Visualization Scripts

| File | Description |
|------|-------------|
| `plot_comparison.m` | Generates comparative plots between ETO-LOS and Backstepping-LOS. |
| `trigger_comparison.m` | Generates inter-event time (IET) stem plots comparing Unified ETM and Channel ETM. |
| `trajectory_comparison.m` | Generates formation trajectory comparison plots. |
| `error_comparison.m` | Generates tracking error comparison plots. |

---

## Usage

Run `main_unified_ETM.m` in MATLAB to reproduce the main simulation results.  
Run `main_channel_ETM.m` to reproduce the Channel ETM comparative results.

## Requirements

- MATLAB R2020a or later  
- No additional toolboxes required
