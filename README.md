# USV-formation-control
Simulation codes for "Event-Triggered Optimal Line-of-Sight Formation Control for Underactuated USVs with Unknown Disturbance"
The full code will be made available upon acceptance/publication of the paper.
For inquiries, please contact: 1732597142@hrbeu.edu.cn

## File Description

### Main Scripts
| File | Description |
|------|-------------|
| `main.m` | Main simulation script for the proposed ETO-LOS scheme. Initializes USV states, runs the two-stage ACI control loop, implements the unified event-triggered mechanism, maps control outputs to actuator commands, and generates all simulation figures. Simulation duration: 300 s, sampling interval: 0.05 s. |
